# Report Agent

## Purpose

Generate professional reports from analysis, forecasts, insights, and recommendations.

---

## Report Types

### Executive Summary

High-level overview.

### Technical Report

Detailed findings.

### Action Plan

Recommended interventions and priorities.

---

## Inputs

* Analytics results
* Forecasts
* Insights
* Recommendations

---

## Outputs

PDF reports and downloadable summaries.

---

## Sections

* Overview
* Key Findings
* Forecasts
* Risks
* Recommendations
* References
