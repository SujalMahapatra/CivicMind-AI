# Analytics Agent

## Purpose

The Analytics Agent is responsible for transforming raw datasets into actionable intelligence.

It performs automated profiling, statistical analysis, trend detection, KPI generation, and anomaly detection.

---

## Inputs

* CSV files
* Excel files
* JSON datasets

---

## Responsibilities

### Data Profiling

Generate:

* Row count
* Column count
* Missing values
* Duplicate records
* Data types

---

### Statistical Analysis

Generate:

* Mean
* Median
* Minimum
* Maximum
* Standard deviation

---

### Trend Detection

Identify:

* Growth trends
* Declining trends
* Seasonal patterns

---

### Anomaly Detection

Detect:

* Outliers
* Sudden spikes
* Sudden drops

---

### KPI Generation

Public Health:

* Health Risk Index

Environment:

* Sustainability Score

Mobility:

* Congestion Score

---

## Outputs

{
"summary": "",
"kpis": {},
"trends": [],
"anomalies": [],
"data_quality_score": 0
}
